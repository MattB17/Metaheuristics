Optimization Solution¶

The OptimizationSolution class acts as a container class for a solution to an optimization problem. It is a convenient object to be used in optimization algorithms in order to aid in the comparison of solutions

class metaheuristics.OptimizationSolution(solution_value, objective_value)[source]¶

A container class for a solution to an optimization problem

Attributes

solution_value: object An object representing the solution itself. For example, in the QAP problem this would be a list, specifying an ordering of the facilities, whereas for the himmelblau problem this would be a tuple specifying the x and y coordinates
objective_value: float The objective value of the solution based on the optimization problem
